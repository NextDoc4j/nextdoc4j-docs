# 网关聚合界面

本页聚焦网关聚合模式下的前端界面能力。

## 1. 服务选择与菜单隔离

### 能力点

- 自动识别聚合模式。
- 顶部/侧边展示服务选择器。
- 按服务维度隔离接口菜单与详情内容。

### 适用场景

- 多微服务统一文档入口。
- 按服务快速切换调试。

![gateway-plugin-ui](/images/screenshots/guide/gateway-plugin-ui.png)
*▲ 聚合模式服务选择器*

## 2. 聚合模式下的调试与检索

### 能力点

- 聚合场景下全局搜索与结果定位。
- 服务切换状态记忆，降低重复操作。
- 聚合模式下在线调试路径自动修正。

## 3. 聚合导出

- 支持按服务、按分组导出。
- 支持微服务聚合文档导出与 `docx`。
